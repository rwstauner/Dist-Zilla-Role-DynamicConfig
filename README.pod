=head1 Dist::Zilla::Role::DynamicConfig

This is a L<Dist::Zilla::Role> for accepting a dynamic configuration,
along the lines of L<Dist::Zilla::Plugin::Prereqs>,
or L<Dist::Zilla::Stash::PodWeaver> (which actually uses this role).

See POD in the modules for more information.

=head1 HISTORY

This repository was forked form the original
L<Dist::Zilla::Stash::PodWeaver> repository
after some of its components were generalized.

=cut
